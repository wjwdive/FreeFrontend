# Free Frontend

基于 Vue3 + Pinia + Vant + JavaScript 的纯前端演示项目

## 🎯 项目特点

- **纯前端演示**: 无需后端服务，开箱即用
- **现代化技术栈**: Vue3 + Pinia + Vant
- **移动端优先**: 响应式设计，完美适配移动设备
- **演示模式**: 模拟用户注册/登录流程

## 🛠 技术栈

- **框架**: Vue 3.4.0
- **路由**: Vue Router 4.2.5
- **状态管理**: Pinia 2.1.7
- **UI组件库**: Vant 4.8.2
- **构建工具**: Vite 5.0.0

## 📁 项目结构

```
src/
├── components/    # 公共组件
│   ├── Loading.vue    # 加载组件
│   └── index.js       # 组件索引
├── router/        # 路由配置
│   └── index.js
├── stores/        # 状态管理
│   └── user.js    # 用户状态
├── views/         # 页面组件
│   ├── Login.vue  # 登录页（演示模式）
│   ├── Register.vue # 注册页（演示模式）
│   └── Home.vue   # 首页
├── App.vue        # 根组件
└── main.js        # 入口文件
```

## 🚀 快速开始

### 方式一：使用HTTP服务器（推荐）

```bash
# 使用任何静态文件服务器
npx serve dist
# 或
python -m http.server 3000 --directory dist
# 或直接打开 dist/index.html
```

### 方式二：开发模式（需要解决依赖安装）

```bash
npm install
npm run dev
```

### 构建生产版本

```bash
npm run build
```

构建后的文件位于 `dist/` 目录。

## 🎮 使用说明

### 演示流程

1. **访问首页**: 自动跳转到登录页
2. **注册账号**: 填写用户名、邮箱、密码进行注册演示
3. **登录系统**: 使用注册的用户名密码登录
4. **查看首页**: 登录成功后进入用户主页

### 功能说明

- **注册功能**: 演示表单验证和用户注册流程
- **登录功能**: 演示用户认证和状态管理
- **状态持久化**: 刷新页面后保持登录状态
- **响应式设计**: 完美适配手机、平板、桌面设备

## 🎨 界面特色

- **渐变背景**: 现代化的渐变色彩设计
- **卡片布局**: 清晰的卡片式界面布局
- **交互动画**: 流畅的页面切换和按钮动画
- **移动优化**: 针对移动设备的触摸优化

## 📱 访问方式

直接打开 `dist/index.html` 或部署到任意静态文件服务器。

## 🔧 技术亮点

- **Vue3组合式API**: 现代化的组件开发方式

- 本项目为纯前端演示，所有数据存储在浏览器本地

- 如需连接真实后端服务，可参考原始代码结构进行扩展
